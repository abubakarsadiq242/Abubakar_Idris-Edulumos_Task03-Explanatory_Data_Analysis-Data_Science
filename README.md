# Abubakar_Idris-Edulumos_Task03-Explanatory_Data_Analysis-Data_Science
This project explores a simulated IPL dataset using Python, Pandas, Matplotlib, and etc. It focuses on data cleaning, trend analysis, and visualization of team performance, player awards, toss impact, match results, and seasonal patterns. The project demonstrates practical skills in data wrangling, exploratory data analysis, and sports analytics.

IPL Data Analysis Project
Project Overview

This project explores a simulated Indian Premier League (IPL) dataset containing match details such as teams, seasons, venues, toss decisions, winners, total runs, wickets, and player of the match awards.

Using Python libraries like Pandas, Matplotlib, and Seaborn, the project focuses on data cleaning, trend analysis, and visualization of IPL performances.

Objectives
Perform data cleaning and preprocessing
Analyze team performance and match trends
Examine toss impact on match outcomes
Identify top players based on awards
Visualize seasonal and venue-based patterns
Gain practical experience in sports data analytics

Tools & Technologies
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
